Week 5

high fidelity prototype

In deze week heb ik gewerkt aan het omzetten van mijn Figma-ontwerpen naar HTML. Hiervoor heb ik gebruikgemaakt van code die gegenereerd werd met behulp van ChatGPT, die ik vervolgens heb aangepast en toegepast binnen mijn eigen project. Tijdens dit proces heb ik geleerd hoe mijn ontwerpen in Figma vertaald kunnen worden naar een werkende website. Ik heb me vooral gefocust op de structuur van de HTML en het correct plaatsen van elementen zoals tekst, knoppen en afbeeldingen, zodat de schermen zo goed mogelijk overeenkwamen met het oorspronkelijke ontwerp. Deze week heeft mij geholpen om beter te begrijpen hoe design en code samenkomen en vormde een belangrijke stap in de realisatie van mijn project.
